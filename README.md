# Otello

Otello, a game written in C. Uses SDL library for UI part. Can be compiled and run on Linux/Windows.

## Dependencies

* libsdl
* libsdl-ttf
* libsdl-image

## Build

```sh
$ make
```

## Usage

```sh
$ ./otello
```

## Keyboard Keys:

* Q - quit program
* B - undo last move
